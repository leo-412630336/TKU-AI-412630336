
## 專案介紹 
此專案為一個實作了多項資訊安全防護機制的 PHP Web 應用程式。主要功能包含使用者註冊、登入驗證、以及依據角色權限（管理員與一般使用者）區分的後台功能。

專案重點在於展示如何防禦常見的網路攻擊，例如：
*   **暴力破解攻擊 (Brute Force)**：實作登入嘗試次數限制與帳號鎖定機制。
*   **惡意行為偵測**：偵測異常操作並自動將使用者登出。
*   **權限控管 (RBAC)**：確保只有管理員能存取特定敏感功能。

系統使用 **PHP** 作為後端語言，並搭配 **MongoDB** 資料庫進行資料存儲。

---


## 1. 環境準備 
確保電腦已安裝以下軟體：
*   **PHP**
*   **MongoDB**

## 2. 啟動步驟 
### 步驟一：啟動資料庫
打開 "命令提示字元 (cmd)" 或 "PowerShell"，輸入以下指令並按下 Enter：
```cmd
mongod
```
### 步驟二：啟動網站伺服器
直接雙擊專案資料夾中的 **`start_server.bat`** 檔案。
它會自動幫您執行 PHP 伺服器指令。

### 步驟三：開啟網頁
打開瀏覽器，網址輸入：
[http://localhost:8000](http://localhost:8000)
