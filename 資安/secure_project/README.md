
## 1. 環境準備 (Prerequisites)
確保電腦已安裝以下軟體：
*   **PHP**
*   **MongoDB**

## 2. 啟動步驟 (How to Start)

### 步驟一：啟動資料庫
打開 "命令提示字元 (cmd)" 或 "PowerShell"，輸入以下指令並按下 Enter：
```cmd
mongod
```
### 步驟二：啟動網站伺服器
直接雙擊專案資料夾中的 **`start_server.bat`** 檔案。
它會自動幫您執行 PHP 伺服器指令。

### 步驟三：開啟網頁
打開瀏覽器，網址輸入：
[http://localhost:8000](http://localhost:8000)
